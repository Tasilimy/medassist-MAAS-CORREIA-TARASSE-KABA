# Projet MedAssist - Migration BDD

## Objectif
Mettre en place l'infrastructure PostgreSQL + Flyway, sans création de tables à ce stade.

## Démarrage
1. docker compose up -d

## Important
1. Le dossier flyway/sql est volontairement vide pour l'instant.
2. Donc aucune table n'est créée tant qu'aucun script de migration n'est ajouté.

## Collaboration Git
1. git add .
2. git commit -m "Init infra Docker + Flyway"
3. git push -u origin main
